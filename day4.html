🗺 11 個札幌日歸選項（自由挑）

① 遠 旭山動物園日歸（推薦 5/26 用 JR Pass）

  • 移動：札幌 → 旭川（特急 Lilac/Kamui 1h25m ¥5,440，JR Pass 免費）→ 旭川駅前搭旭川電氣軌道 41/47 號巴士 40 分 ¥500 → 旭山動物園
  • 票價：旭山 ¥1,000（外國觀光客）
  • 必看：企鵝散步（11:00, 14:30）、北極熊、海豹館
  • 建議時間：09:00 出發 → 18:00 回札幌
  • 套票替代：KKday 旭山+青池+白鬚瀑布一日遊 約 ¥8,000（中文導遊，5 月青池仍冰未化但仍藍）

② 中 洞爺湖日歸 KKday 11257 一日遊

  • 套票：KKday 札幌出發 洞爺湖+登別+昭和新山一日遊
  • 價格：約 ¥10,000-12,000/人 · 中文導遊
  • 行程：札幌 8:30 → 昭和新山 → 洞爺湖展望台 → 登別地獄谷 → 札幌 18:30
  • 適合：不想坐 JR、想看火山+溫泉地獄
  • 注意：5/26 二平日場次可能取消，要提前 3-5 天訂

③ 近 小樽日歸（JR 30 分）

  • 移動：札幌 → 小樽（JR 快速 32 分 ¥800，JR Pass 涵蓋）
  • 路線：三角市場早餐 → 運河 → 堺町通り（北菓樓/六花亭/LeTAO）→ 政壽司/魚真壽司 → 天狗山纜車（夜景）
  • 必吃：滝波食堂海鮮丼、政壽司握壽司、LeTAO 雙層起司蛋糕
  • 適合：必訪打卡、CP 值高

④ 近 余市威士忌酒廠+小樽（JR Pass 划算）

  • 移動：札幌 → 余市（JR 快速空知或普通 1h15m ¥1,380，JR Pass 涵蓋）
  • 必看：Nikka 余市蒸餾所（免費見學 9:00-15:30，最終受付 15:00）· 朝日電視劇《阿政》取景地
  • 回程：余市 → 小樽（JR 25 分），順遊小樽
  • 適合：威士忌迷

⑤ 市區 北海道神宮+圓山公園+丁香花祭

  • 移動：地鐵東西線「圓山公園」站（¥210）或 JR 巴士西 14/15 線「神宮前」站（飄兒實測更近）
  • 路線：神宮參拜 → 公園散步 → 丁香花樹（5/20-31 開花期）
  • 適合：文化派、季節派

⑥ 市區 白色戀人公園+藻岩山纜車（夜景）

  • 移動：地鐵東西線「宮の沢」站步行 7 分 → 公園 ¥800 入園
  • 必體驗：白色戀人巧克力觀光工廠、玫瑰花園、限定品「白色戀人巧克力牛奶」（只有公園販賣機買得到）
  • 晚上：藻岩山纜車（市電轉接駁巴士）¥2,100 看夜景
  • 適合：親子、夜景控

⑦ 市區 純札幌精華（北大+二条市場+狸小路）

  • 路線：北 12 條站 → 北大白楊木林 → 札幌駅 → 二条市場午餐 → 狸小路 → 大通公園 → 札幌啤酒博物館（免費）
  • 適合：累、雨天、想慢慢逛

⑧ 中 支笏湖日歸（巴士 1h）

  • 移動：札幌駅前 → 中央巴士「支笏湖線」1h ¥1,050（JR Pass 不涵蓋）
  • 必玩：支笏湖透明度日本第 1、清水濱湖步道
  • 適合：純自然派、想看湖

⑨ 遠 定山溪溫泉日歸（巴士 1h）

  • 移動：札幌駅前 → じょうてつ巴士「定山溪溫泉」1h ¥790（JR Pass 不涵蓋）
  • 必體驗：鹿の湯日歸入浴 ¥1,500、二見吊橋散步
  • 適合：想泡湯

⑩ 近 三井 Outlet 北廣島

  • 移動：札幌駅 → 北廣島駅（JR 17 分 ¥580）→ 接駁巴士 9 分
  • 200+ 品牌、北海道最大 Outlet
  • 適合：購物日

⑪ 市區 円山動物園 + 六花亭神宮茶屋

  • 地鐵「圓山公園」步行 15 分 → 円山動物園 ¥800（猛獸館 2023 新開）
  • 順遊神宮、六花亭神宮茶屋限定「判官さま」
