🔍 Day 7（5/29 五）你有 4 個選擇

「整天自由 + 5/29 起 ES CON FIELD 有主場賽（vs 巨人交流戰）」

⚾ 5/29 五 18:00 開賽：日本火腿 vs 讀賣巨人，在 ES CON FIELD（北廣島，從札幌站快速エアポート 17 分 ¥580 + 接駁巴士 ¥200）
👉 訂票：Fighters 官網

🅰️ A 案：小樽半日 + 看球（推薦給愛新鮮體驗）

適合：看小樽運河走馬看花 + 體驗日本職棒

預算：¥3,500（JR + 接駁）+ 球票 ¥3,000-8,000

  • 🕒 08:30 JR 札幌 → 小樽 ¥800
  • 🕒 09:10 三角市場早午餐（滝波食堂 / 味処たけだ）
  • 🕒 10:30 小樽運河 + 堺町通商店街（LeTAO 3F 觀景 + 茶和々抹茶 + 北一硝子星空喫茶）
  • 🕒 13:00 魚真壽司午餐
  • 🕒 15:30 田中酒造試飲
  • 🕒 16:30 JR 回札幌
  • 🕒 17:15 快速エアポート → 北廣島站 ¥580（17 分）
  • 🕒 18:00 開賽 火腿 vs 巨人 ⚾
  • 🕒 21:30 比賽結束，臨時列車回札幌

✅ 優：小樽精華+球賽，一天兩大體驗

❌ 缺：小樽走馬看花、球賽要訂票（5/29 五場最熱門）

🅱️ B 案：純小樽深度全日（不看球）

適合：愛小樽運河童話世界 + 不愛運動

預算：¥2,500+

  • 🕒 09:00 JR 札幌 → 小樽（不趕，吃飽再走）
  • 🕒 10:00 三角市場早餐 + 海鮮丼
  • 🕒 11:00 小樽運河慢拍照
  • 🕒 12:00 堺町通 + 大正硝子館 + 北一硝子（玻璃工坊體驗 ¥1,500）
  • 🕒 13:30 政壽司 / 魚真壽司高級午餐 ¥5,000+
  • 🕒 15:00 田中酒造試飲 + 龜甲蔵見學
  • 🕒 16:30 天狗山纜車 ¥1,400（小樽夜景）
  • 🕒 18:30 LeTAO 運河廣場店晚餐+甜點酒吧（小樽第一間）
  • 🕒 21:00 JR 回札幌

✅ 優：小樽完全深度，包含夜景

❌ 缺：來回 JR 多次累、錯過 ES CON 主場機會

🅲 C 案：余市 NIKKA + 小樽 + 看球

適合：愛威士忌 + 喜歡多去一個城市

預算：¥3,000+ + 球票

  • 🕒 08:30 JR 札幌 → 余市（先到小樽再換普通車，55-65 分，¥1,270）
  • 🕒 10:00 余市 NIKKA 蒸餾所（免費見學 90 分 + 試飲 3 杯，需預約）
  • 🕒 12:00 柿崎商店海鮮工房午餐（便宜海鮮丼）
  • 🕒 13:30 JR 回小樽，逛運河 + 堺町通
  • 🕒 16:00 JR 回札幌
  • 🕒 17:15 快速エアポート → 北廣島看球
  • 🕒 18:00 開賽

✅ 優：3 個城市一天串完（小樽+余市+北廣島）

❌ 缺：很趕、NIKKA 需事先預約（3 個月前）

🅳 D 案：不去小樽，札幌深度 + 看球

適合：不愛小樽 / 怕走馬看花 / 想札幌玩透

預算：¥2,000+ + 球票

  • 🕒 09:00 飯店悠閒早餐
  • 🕒 10:00 北海道大學散步 + 綜合博物館（免費）
  • 🕒 12:00 大丸 B1 KINOTOYA + Fruitscake Factory 午餐
  • 🕒 14:00 啤酒博物館（11-18，週一休 → 週五 OK）
  • 🕒 16:00 莫埃來沼公園 野口勇玻璃金字塔（東豐線+巴士）
  • 🕒 17:15 從中央區搭快速エアポート → 北廣島
  • 🕒 18:00 開賽 火腿 vs 巨人

✅ 優：札幌大師作品（北大、啤酒博、莫埃來沼）一網打盡

❌ 缺：沒去小樽（這趟唯一機會）

🤔 我的推薦

選 A 半日小樽+看球：9 成人選這個。小樽運河 32 分鐘就到，半天看完精華；ES CON 5/29 五場是主場黃金場，錯過要等明年。

選 D 札幌深度+看球：如果你完全不愛小樽（覺得是觀光客陷阱）— 飄兒實測「小樽走馬看花 OK，深度比札幌淺」。

選 B / C 純小樽 / 加余市：放棄球賽，但球賽 5/30 六、5/31 日還有（5/31 因 13:00 開賽 + 你 16:15 飛機看不完，所以實際只剩 5/30 補球）。
